# Emre Koc - Website

Personal portfolio website showcasing my professional experience and technical skills.

**Visit live:** [emrekoc.io](https://emrekoc.io)

## Features

-   **Responsive Design** - Seamlessly adapts to all device sizes
-   **Animations** - Content reveal transitions and interactive elements
-   **Performance Optimized** - Built with Next.js for SSR and SEO
-   **Analytics Integration** - Google Analytics for visitor insights
-   **Type-Safe** - Fully typed with TypeScript
-   **Modern Styling** - Styled Components for maintainable CSS-in-JS

## Tech Stack

-   **Framework:** [Next.js](https://nextjs.org/)
-   **UI Library:** [React](https://react.dev/)
-   **Styling:** [Styled Components](https://styled-components.com/)
-   **Language:** [TypeScript](https://www.typescriptlang.org/)
-   **Icons:** [Font Awesome](https://fontawesome.com/)
-   **Analytics:** [Google Analytics](https://analytics.google.com/)

## Project Structure

```
src/
├── app/              # Next.js app directory
├── components/       # Reusable UI components
├── contexts/         # React context providers
├── sections/         # Main page sections
└── types/            # TypeScript type definitions
```

## Key Sections

-   **Intro** - Landing section with animated introduction
-   **Tech** - Technical skills and technologies
-   **Experience** - Professional work history
-   **Education** - Academic background
-   **Connect** - Contact information and social links

## Development Highlights

-   **Component-Based Architecture** - Modular, reusable components
-   **Context API** - For analytics and page transitions
-   **SEO Optimized** - Meta tags and Next.js optimizations
-   **Accessibility** - Semantic HTML

## Getting Started

### Prerequisites

-   Node.js 22.x or higher
-   Yarn package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/Emroni/website.git

# Navigate to the project directory
cd website

# Install dependencies
yarn
```

### Development

```bash
# Start the development server
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site in your browser.

### Build

```bash
# Create a production build
yarn build

# Start the production server
yarn start
```

### Linting

```bash
# Run ESLint
yarn lint
```
